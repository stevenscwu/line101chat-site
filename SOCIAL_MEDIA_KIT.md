## Profile Photo Guidance

Primary recommendation:

- Use `public/presenter/host-main.png` if the face remains clear after a square crop.
- Backup option: `public/presenter/1.png` for warmer contact-oriented presence.
- Homepage consistency option: `public/presenter/3.png` if the social account should match the website hero.

Crop style:

- Square 1:1.
- Face centered.
- Chest-up or shoulder-up.
- Background clean and bright.
- Add small emerald or LINE green border only if it remains professional.

Do not:

- Use every presenter image as a rotating profile identity.
- Use a busy background.
- Add large text inside the circular avatar because it becomes unreadable.
