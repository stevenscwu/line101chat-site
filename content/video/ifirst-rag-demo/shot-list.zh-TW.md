# Shot List｜北科大創新學院 RAG Demo 影片

建議總長：60-90 秒

## 必拍畫面

1. LINE101Chat 首頁或服務頁
   - 顯示「AI 知識助理」與免費評估 CTA。
   - 用於 Scene 2 與 Scene 5。

2. Case studies page
   - 顯示北科大創新學院 iFIRST AI 文件問答 Demo 區塊。
   - 顯示影片區與 QR code guide。

3. NTUT iFIRST Demo chatbot QR
   - 使用 `public/ntut-ifirst-demo-qr.png`。
   - 標示為「試用北科大創新學院文件問答 Demo」。

4. LINE demo interaction
   - 開啟北科大創新學院 Demo chatbot。
   - 輸入：「人工智慧學程 TOEFL iBT 畢業門檻是多少？」
   - 顯示回答與資料來源。
   - 不顯示私人 LINE ID、個人訊息或通知內容。

5. Value cards
   - 減少重複問答
   - 保留組織知識
   - 新生 / 新員工更快上手
   - 回答可追溯來源
   - 支援 LINE 與網站

6. Business inquiry CTA
   - 使用 `public/line101-business-qr.png`。
   - 標示為「詢問 LINE101Chat 服務」。
   - 搭配「免費評估」按鈕。

## 遮蔽與安全

- 錄製 LINE 畫面前，開啟勿擾模式。
- 避免顯示私人 LINE ID、群組名稱、通知或其他聊天內容。
- Demo 只展示公開文件問答，不展示學生個人資料。
- 若錄到瀏覽器收藏列或本機路徑，可在剪輯時裁切或模糊。
