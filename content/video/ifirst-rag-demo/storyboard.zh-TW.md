# Storyboard｜北科大創新學院 iFIRST AI 文件問答 Demo

影片副標：用公開文件建立可在 LINE 查詢的 AI 知識助理  
影片長度：60-90 秒  
比例建議：16:9 為主，可另裁切 9:16 短影音版本。  
語氣：專業、可信任、務實、台灣商務友善。

| 時間 | 場景 | 畫面建議 | 旁白重點 | 螢幕文字 |
|---|---|---|---|---|
| 0-5 秒 | Problem | 學生 LINE 訊息、規章 PDF、表單資料夾 | 學生常問畢業門檻、資格考、表單、修業規定 | 學生反覆詢問 |
| 5-10 秒 | Repeated Work | 辦公室回覆、系所網站、FAQ 文件 | 學校與系所常需要重複回答 | 窗口重複回答 |
| 10-15 秒 | Solution | LINE101Chat case-studies 頁面與 AI 知識助理服務 | 把正式文件整理成 AI 知識助理 | AI 知識助理 |
| 15-20 秒 | LINE Entry | LINE 提問示意、文件檢索流程 | 使用者直接在 LINE 提問，AI 先查文件 | LINE 查詢 / 來源引用 |
| 20-28 秒 | Demo Setup | 北科大創新學院 iFIRST Demo chatbot QR 或聊天畫面 | Demo 使用公開文件建立，非官方服務 | iFIRST RAG Demo |
| 28-38 秒 | Demo Question | 在 LINE 輸入：「人工智慧學程 TOEFL iBT 畢業門檻是多少？」 | 直接用自然語言提問 | 直接用 LINE 提問 |
| 38-50 秒 | Demo Answer | 顯示 bot 回答與來源資訊 | Bot 從官方公開文件查找內容並整理回答 | AI 先查文件，再根據來源回答 |
| 50-55 秒 | Disclaimer | 放大來源欄位或提示文字 | 正式規定仍以官方公告與辦公室回覆為準 | 來源可追溯 |
| 55-64 秒 | More Questions | 例題卡片輪播 | 資格考、修業規定、同意書、半導體學程規定 | 更多文件問題 |
| 64-70 秒 | More Use Cases | 招生 FAQ、SOP、產品手冊、內部規章、客服知識庫 | 同樣方法可用於學校與企業文件 | 可複製到組織知識 |
| 70-80 秒 | Free Assessment | 免費評估頁或 CTA 區塊 | 先用 20-30 頁正式文件評估 | 20-30 頁正式文件 |
| 80-90 秒 | CTA | line101chat.com、商務詢問 QR、免費評估按鈕 | 前往網站、預約免費評估、加入 LINE 詢問 | line101chat.com |

## 視覺原則

- 使用網站現有的 emerald 綠、slate 中性色、白底與深色影片區塊。
- 畫面要像商務 Demo，不使用過度誇張的科技動畫。
- iFIRST Demo chatbot 只標示為「試用北科大創新學院文件問答 Demo」。
- LINE101Chat 商務帳號只標示為「詢問 LINE101Chat 服務」。
- 避免露出 LINE Developers Console、後台網址、私人 LINE ID、通知或其他聊天內容。
