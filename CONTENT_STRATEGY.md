# CONTENT_STRATEGY.md

## Mission
Build a long-term, low-friction Japanese learning content engine for a Taiwanese 50+ learner.

## Content Pillars
1. Daily mission progress
2. Pronunciation and speaking clips
3. Vocabulary and phrase growth
4. Honest reflection and challenge logs
5. Weekly momentum snapshots

## Social Output Templates
1. 今日一句（Japanese + zh-TW）
2. 今日卡關點（what was hard and why）
3. 一週小結（progress + next focus）
4. 跟讀挑戰（repeat-after-me clip）
5. 50+ 學習者反思（warm and realistic）

## Copy Principles
- concise
- emotionally warm
- practical
- non-corporate
- non-childish

## UI Language Hierarchy
- UI: zh-TW first
- Learning content: Japanese first
- Supporting explanation: zh-TW

## Reusability Direction
- Keep learning content types strongly typed
- Keep locale strings separate from learning material
- Allow future user template cloning without changing app shell
